File size must not exceed 1024 * 1024 * 1024 size (set to be able to upload almost all files) otherwise the system will display Gray Blue Screen of Death (GBSOD) ;)

Example 1:

you have uploaded file named "test.jpg", but you have populated File name: text field with name test_upload (note: no extension) - resulting file will have name: "test_upload.jpg"

Example 2:

you have uploaded file named "test_initial_name.jpg", but you have left the File name: text field empty - resulting file will have name: "test_initial_name.jpg"
